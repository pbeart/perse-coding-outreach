This is a sample lesson, which uses Blocks programming on the micro:bit to teach repetition and iteration

Materials needed

  • BBC micro:bit (optional)
  • A laptop or computer

Learning objectives

  • Learn how you can repeat a block of instructions in MakeCode
  • Understand what you can use repetition for
  • Understand the idea of iteration, including stopping a loop when a condition is met

Code Example 1: The Repeat Block

Repeat block

In this example, students can see how blocks can be executed multiple times using a Repeat block.

Suggestions

  • We encourage you to let students guess how many times the face will appear and disappear, then count along.
  • Then, try changing the repeat value, and again ask students what they think will happen.
